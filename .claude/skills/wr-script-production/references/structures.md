# Script structures (SK-B3)

## The 7-beat 90-second skeleton (the reel canon)

```
[0:00 HOOK]         One breath. Earns the next five seconds.
[0:05 STAKES]       Why this matters to this specific person. Name them.
[0:15 BEAT ONE]     The thing they think is true.
[0:35 BEAT TWO]     The thing that's actually true.
[0:55 BEAT THREE]   The thing nobody told them.
[1:15 TURN]         The line that reframes the whole video.
[1:25 CTA]          One ask. Never two.
```

The TURN must genuinely reframe — a restatement scores 1 on logical flow
(RUB-SCRIPT-1). The CTA is a natural exit from the turn, zero pressure
language.

## Long-form / webinar variant

Welcome → agenda → disclaimer (read aloud — a compliance moment, not a
throwaway) → founder beat → teaching sections (each: claim → concrete picture
→ stress-relief beat → one-line takeaway) → mistakes section → decision
framework → recap → resources → soft CTA. The approved DROP Exit webinar
(assets/webinar-scripts/01) is the worked example.
