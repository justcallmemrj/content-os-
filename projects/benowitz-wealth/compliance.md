# Benowitz compliance rules (PROTECTED)

**The firm-wide envelope applies in full:**
@projects/_shared/ria-compliance-envelope.md

## Brand-specific rules

- **BEN-C1 — FRS non-affiliation on every piece.** Benowitz Wealth Management is
  not affiliated with, endorsed by, or sponsored by the Florida Retirement
  System, the Florida Division of Retirement, or the State Board of
  Administration. Public employees will reasonably assume otherwise if unsaid;
  silence is an implied claim. Verbatim texts: `disclosures.md`.
- **BEN-C2 — Pension-solvency fear is off-limits** as a persuasion device —
  contested, political, corrosive to trust with this audience.
- **BEN-C3 — Never imply FRS wrongdoing or concealment.** FRS administers a
  pension; it doesn't do financial planning. Saying so plainly requires no villain.
- **BEN-C4 — Never suggest Benowitz can improve the pension itself.** The
  pension is the pension; what's decidable is everything around it.
- **BEN-C5 — Tax content carries the tax referral disclosure** (DISC-BEN-TAX-01
  appended to DISC-BEN-FRS-01).
- **BEN-C6 — "fee-only" hold** per DEC-BEN-0001; use "fiduciary."

## Disclosure mapping

| Context | Required |
|---|---|
| Any published piece | DISC-BEN-FRS-01 (verbatim, written out — never "[add disclaimer]") |
| Touches taxes | + DISC-BEN-TAX-01 |
| Space-constrained (Story frame, Reel cover) | DISC-BEN-SHORT-01 minimum + full text in profile/first comment, stated explicitly |
| Video | Persistent on-screen block, not the spoken track |
| LinkedIn | Full disclosure in first comment; the post states it's there |
